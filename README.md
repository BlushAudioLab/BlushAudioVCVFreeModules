**Buff Mix**

A clone of the Blush Audio Buff Mixer.

6:1 or 2x 3:1 Unity Mixer - Output 1 is normalled to Output 2.

A clone of the Blush Audio Buff Mult.
 
1:6 or 2x 1:3 Multiple - Input 1 is normalled to Input 2.
